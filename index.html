Send raw command

Enter the payload only (command byte + params, hex). The FE FE, length and checksum are added automatically. Example 01 = query, 05 ec = set −20°C.

⚠️ The fridge accepts commands with no authentication. Don't send random bytes to 02 (set-all) or 04 (reset) unless you know the layout — you could change settings or factory-reset it.

About this build

Protocol: Alpicool / Brass Monkey frame FE FE | len | payload | sum16. Service 1234, write 1235, notify 1236. The frame/checksum core is verified against the official CAR FRIDGE FREEZER app; the per-byte status map is best-effort and may differ on your 50 L — help confirm it with the Snapshot tool above. See PROTOCOL.md in the repo.
